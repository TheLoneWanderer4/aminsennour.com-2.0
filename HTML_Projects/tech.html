Python Lists

Before we actually start talking about linked lists it's important to understand how traditonal python lists are implemented in memory.

In python, when you create a list alist = [10] a python list object is added to memory. Since this list is created with a single element, 10, it's represented in memory as a single memory slot with "10" followed by a number of empty memory slots. When you grow the list alist.append(11) the added value, in this case 11, is added to one of those reserved memory slots.

This allows for two benefits.

  • The size of a python list can be changed after it's creation. This isn't true in many other languages.
  • Indexing the list is extremely simple, you just add the index to the memory location of the start of the list.
However, it also comes with trade-offs.
  • Python lists are cumbersome and taxing on memory, because they require extra space.
  • Adding an element to a position that's not the end of a list alist.insert(0, 9) is O(n) because it requires every single element to be relocated

For the vast majority of situations in python this is the optimal balance, but, in your own programming career, you may come up against a problem where this is not the case. Have no fear, for LinkedLists are here! and they solve many of these problems.

What are Linked Lists?

Linked lists are an implementation of lists which consists only of a lists most basic elements, a value at a memory location, and the memory location of the next value

This is normally represented by two classes

  • a Node class, which only consists of a value, and a pointer to the next value
  • a Linkedlist class, which consists of the first node, as well as helper function to add functionality

Why Linked Lists?

This implementation removes two of python lists primary faults.

  • The use of pointers to guide the tract of the list allow for the elements to be placed totally randomly in memory, resulting in very low usage.
  • The nature of each element pointing only to it's direct heir allows for elements to be inserted at any spot easily just by changing where the element before that spot points to.

The linkedlist class - 


          class LinkedList:
              def __init__(self):
                  ''' init '''
                  self._head = None

              def add(self,new):
                  new = Node(new)
                  '''
                  simple method to add a new node, new, to the front of self
                  Paramiters: new, a node
                  Return: none
                  Pre_condition: new is of the Node class
                  Post_condition: new is now the first node in self
                  '''
                  new._next = self._head
                  self._head = new

              def append(self, new):
                  new = Node(new)
                  '''
                  method to add a new node, new, to the end of a linked list
                  Paramiters: New, a node
                  Return: None
                  Pre-condition: New is a node
                  Post-condition: New is at the end of self
                  '''
                  curr = self._head
                  if curr == None:
                      self._head = new
                      return
                  while curr._next != None:
                      curr = curr._next
                  curr._next = new

              def __str__(self):
                  ''' str '''
                  n = self._head
                  ret = ''
                  while n!= None:
                      ret += str(n) + '\n'
                      n = n._next
                  return ret.strip()
